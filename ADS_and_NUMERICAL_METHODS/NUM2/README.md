Krótki opis: 
W pliku znajdziemy program obliczajacy uklady rownan z zaburzeniem i bez dla dwoch macierzy
z uzyciem biblioteki numpy, by przeanalizowac jak uwarunkowane numerycznie sa obie macierze.

Jak odpalic program w terminalu: 

python3 main.py

Dodatkowe informacje: 
- program napisany w jezyku Python
- programy daje nam wyniki podpisane kolejno przed dana wartoscia
- program napisany po angielsku z przyzwyczajenia zawodowego
