Krótki opis: 
W pliku znajdziemy2 programy wyliczajace bledy |Dhf(x) - f'(x)| jeden do glownej czesci zadania i drugi do dodatkowej innej funkcji w oryginalnym z zadania i dodatkowym punkcie 

Jak odpalic programy w terminalu: 

python3 main.py - dla glownej czesci zadania 

python3 other.py - dla dodtkowego przykladu i innego punktu 


Dodatkowe informacje: 
- program napisany w jezyku Python
- programy tworza łącznie 6 wykresów błędu |Dhf(x) - f'(x)|
- program napisany w pełni po angielsku z przyzwyczajenia zawodowego
