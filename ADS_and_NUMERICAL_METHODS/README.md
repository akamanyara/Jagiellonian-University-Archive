# ADS and Numerical Methods

This repository contains a collection of **algorithmic implementations** and **numerical methods** developed for academic and self-learning purposes.  
The goal of this project is to demonstrate a practical understanding of **data structures**, **graph algorithms**, and **numerical computation techniques** used in computer science and applied mathematics.

---

## 📚 Contents

### Algorithms and Data Structures (AISD)
| Folder | Description |
|--------|--------------|
| `AISD_AVL` | Implementation of AVL (self-balancing) binary search trees |
| `AISD_BST` | Classic Binary Search Tree with insertion, deletion, and traversal |
| `AISD_Dijkstra` | Dijkstra’s shortest path algorithm for weighted graphs |
| `AISD_FLOYD` | Floyd–Warshall algorithm for all-pairs shortest paths |
| `AISD_HASH` | Hash table implementations with collision handling |
| `AISD_KONKURS` | Competition tasks and custom algorithmic problems |
| `AISD_KRASK_PRIM` | Kruskal and Prim algorithms for minimum spanning tree |
| `AISD_LISTS` | Singly, doubly, and circular linked list implementations |
| `AISD_SORT` | Common sorting algorithms (QuickSort, MergeSort, HeapSort, etc.) |
| `AISD_SPLAY` | Splay tree implementation (self-adjusting binary search tree) |

### Graph and Search Algorithms
| Folder | Description |
|--------|--------------|
| `ASTAR` | A* pathfinding algorithm implementation |
| `GRAPH_EC` | Graph exploration and connectivity algorithms |
| `EC` | Miscellaneous algorithmic exercises and experiments |

### Numerical Methods (NUM)
| Folder | Description |
|--------|--------------|
| `NUM1` – `NUM5` | Implementations of numerical techniques such as interpolation, root finding, matrix operations, numerical integration, and differential equation solving |
