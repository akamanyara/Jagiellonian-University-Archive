Krótki opis: 
Zadanie polega na zbadaniu wpływu doboru parametrów do obliczenia układu równań przy uzyciu 
metod iteracyjnych. Sprawdzamy tez zbieznosc tych metod w praktycznym zastosowaniu.

Jak odpalic programy w terminalu: 

python3 main.py - dla glownej czesci zadania 

Dodatkowe informacje: 
- program napisany w jezyku Python
- zastosowana biblioteka numpy w celu ulatwienia obliczen na macierzach
- program napisany w pełni po angielsku z przyzwyczajenia zawodowego
