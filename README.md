# Jagiellonian University Archive

An archive of exercises, algorithms, and side projects created during my Applied Computer Science studies at Jagiellonian University. 

I consolidated my academic coursework into this single repository to maintain a clean GitHub profile, keeping my main page focused on production-ready code and full-stack applications.

---
*Note: Some of my collaborative academic projects are not included here, as they are hosted in separate team or organization repositories.*

--- 
## Repository Contents:
- **ADS_and_NUMERICAL_METHODS** -> Algorithmic and numerical method implementations.
- **Project-Manager** -> Frontend + Backend final project using **Node.js**.
- **PythonUJ25** -> Class exercises covering core **Python** concepts.
- **StudyMateAi** -> Intelligent educational chatbot utilizing **Python, Streamlit, and the OpenAI API**.
- **Tower-Defence** -> Classic tower defense game built with **PyGame**.
- **WWWTechniques** -> Collaborative assignments for WWW Techniques Class **(HTML, CSS & JS)**.
- **aplikacja_kulinarna_cookcat** -> Collaborative project focused on **UI/UX and Graphic Interfaces**.
- **weatherAPI** -> Desktop weather application built with **Python and the PyQt5 Framework**.
- **web-scraper-visualizer** -> Data extraction tool built using the **Selenium** library.
- **graphics** -> Graphics course with **PostScript**, etc.
